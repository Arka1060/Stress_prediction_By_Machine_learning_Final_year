# Stress_prediction_By_Machine_learning_Final_year
stress detection system using TF-IDF and Multinomial Naive Bayes, trained on Reddit data. Streamlit web app with real-time predictions, confidence scores, and keyword explanations.
Step 1 — Install dependencies

python -m pip install scikit-learn pandas numpy matplotlib seaborn streamlit

Step 2 — Train the model

python main.py

Step 3 — Run the app

python -m streamlit run app.py

Step 4 — Open in browser

http://localhost:8501
